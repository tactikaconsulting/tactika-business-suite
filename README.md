# Táctika Business Suite

Sistema ERP/CRM desarrollado por **Táctika Consulting**.

## Características

- Dashboard Ejecutivo
- CRM Clientes
- Gestión RRHH
- Diagnósticos
- Planes de Acción
- Seguimiento
- Reportes
- LocalStorage
- Responsive
- React + Vite

## Tecnologías

- React
- Vite
- Tailwind CSS
- React Router
- Chart.js
- Lucide Icons

## Instalación

```bash
npm install
npm run dev
```
